This project is an interactive Meta (Facebook & Instagram) Ad Performance Dashboard built using Power BI. It provides a comprehensive overview of advertising performance, enabling data-driven decision-making through clear visual insights.

The dashboard analyzes key marketing metrics such as impressions, clicks, engagement, conversions, and budget utilization across different dimensions like age group, gender, ad type, and time.

🚀 Key Features
📈 KPI Cards for quick insights (Impressions, Clicks, CTR, Conversion Rate, Budget)
👥 Audience Analysis by Gender and Age Group
📊 Performance Trends (Hourly & Weekly)
🎯 Ad Type Analysis (Image, Video, Carousel, Stories)
🌍 Country-wise Performance (Treemap visualization)
🗓️ Interactive Monthly Calendar Filter
🔄 Dynamic Measure Selection using slicer

🛠️ Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Data Modeling
Power Query (ETL)

📌 Insights Derived
Identified top-performing ad types and audience segments
Analyzed engagement and conversion trends over time
Compared campaign effectiveness across demographics
